Basic structure of a sketch:

  • Setup
  • Draw

Setup

You have to write this at the beginning before you 'draw'


							// This is a single-line 'comment'. It will NOT disrupt your program.
							/* This is a multi-line comment.
							This will also not disrupt your program. You can include as many comments as you like.
							These help you in understanding your code in more human-like language. */
							
							// This is my setup. Setup is "always" written first in p5.js
							function setup(){
							  createCanvas(400, 400);
							}

Draw

You have to write this after the 'setup'


							/* Draw is always written after the "setup" */
							// draw runs at about 60fps. 'fps' is frames per second

							// This is my draw
							function draw(){
							  //make a background with a color-value of 240
							  background(240);
							  
							  //take a color of rgb value of 235, 40, 95, where these specify red, green, blue, respectively.
							  //the minimum in rgb by default, is 0 and the max is 255
							  fill(235, 40, 95);

							  //make a circle at point x=200, y=200 of radius=20
							  circle(200, 200, 20);

							  //make an ellipse at point x=100, y=300 of a=20, b=20
							  ellipse(100, 300, 20, 20);

							  //make an ellipse at point x=300, y=300 of a=10, b=30
							  ellipse(300, 300, 10, 30);

							  //make an ellipse at point x=500, y=800 of a=50, b=50
							  ellipse(500, 800, 50, 50);
							}

							  background(255);

Conditionals - if/ else-if /else

If(condition is true){do this} else if(this condition is true){do this} else{finally do this}


							function draw(){
							  background(255);
							  if(x > 300){
							   fill(255,100,240);
							  }
							  else if(x < 100){
							   fill(30,200,165);
							  }
							  else{
							   fill(30); 
							  }
							  circle(x, 100, 50);
							}

Conditionals - Ternary

condition ? expr1 : expr2


							function draw(){
							  background(255);
							  (x > 300) ? fill(240) : fill(50);
							  circle(x, 100, 50);
							}

📒 JavaScript Basics

Conditionals - Switch

switch, check cases and execute appropriately


							function draw(){
							  background(255);
							  switch(x){
							   case 10:
								fill(50);
								break;
							   case 50:
								fill(100);
								break;
							   case 150:
								fill(150);
								break;
							   default:
							    fill(240); 
							  }
							  circle(x, 100, 50);
							}
